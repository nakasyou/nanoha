# NanohaNote
## NanohaNoteとは？
一言で言えば、デジタル赤シートです。
二言で言えば、「じぶん」でつくる、暗記用デジタルノートブックです。
あなたの暗記をサポートします！
3.2.0
## 開発
### Develop using Deno
開発モード:
```shell
deno task dev
```
ビルド:
```shell
deno task build
```
### Develop using Node.js
パッケージマネージャは、[pnpm](https://pnpm.io)を用いています。

開発モード:
```shell
pnpm dev
```
ビルド:
```shell
pnpm build
```
### Production
ビルドした後に、
```shell
deno task start
```
です。
